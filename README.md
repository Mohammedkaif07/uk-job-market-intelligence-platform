# UK Job Market Intelligence Platform

## Overview

An end-to-end job market analytics platform built using Python, PostgreSQL, SQL, Power BI, and live job APIs to analyse UK hiring trends, salary patterns, skills demand, and remote work opportunities.

## Business Problem

Job seekers and analysts often struggle to understand:

* Which skills are most in demand
* Which locations have the highest hiring activity
* Current salary trends
* Remote vs on-site opportunities

This project transforms raw job market data into actionable insights through automated data pipelines and interactive dashboards.

## Tools & Technologies

* Python
* PostgreSQL
* SQL
* Power BI
* JSearch API
* Pandas
* NumPy

## Project Workflow

Live Job API → Data Collection → Data Cleaning → PostgreSQL Database → SQL Analysis → Power BI Dashboard

## Key Features

* Automated ETL pipeline
* Live job market data collection
* Salary intelligence analysis
* Skills demand analysis
* Hiring location insights
* KPI reporting dashboard
* Real-time job market feed

## Dashboard Pages

### Executive Overview

Provides a high-level summary of hiring activity, salaries, and job market trends.

### Skills Intelligence

Analyses the most demanded skills, highest-paying skills, and skill demand across job roles.

### Live Job Market Feed

Displays real-time job postings, hiring locations, employment types, and remote opportunities.

## Key Skills Demonstrated

Data Analysis, SQL, PostgreSQL, Power BI, ETL Pipelines, Dashboard Development, Business Intelligence, API Integration, Data Visualisation, Reporting.
